# 文档规范

## 文档编号规则

所有文档都使用 **2位数序号** 编号，例如：`01`, `02`, `03`...

## 存放位置

- 根目录：`docs/`
- 分类目录：`docs/XX-分类名/`
  - ES5 相关 → `docs/01-es5/`
  - ES6 相关 → `docs/02-es6/`

## 分类目录序号

| 序号 | 分类 |
|------|------|
| 01   | es5  |
| 02   | es6  |
| 03   | node |
| 04   | react |
| ...  | ... |

## 文档命名格式

```
docs/XX-分类名/YY-文档名.md
```

- `XX`：分类序号（01, 02, 03...）
- `YY`：该分类下的文档序号（01, 02, 03...）

## 示例

```
docs/
├── 01-es5/
│   ├── 01-javascript-debugging-guide.md
│   ├── 02-javascript-prototype.md
│   └── 03-vscode-shortcuts.md
│
├── 02-es6/
│   ├── 01-es6-new-features.md
│   └── 02-promises.md
│
└── 03-node/
    └── 01-node-basics.md
```

## 新增文档步骤

1. 确定文档所属分类
2. 查看该分类目录下现有文档的最大序号
3. 新文档序号 = 最大序号 + 1（保持两位数，如 01, 02）
