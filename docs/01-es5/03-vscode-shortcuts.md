# VS Code 常用快捷键

## 运行 JavaScript 文件

### 1. 终端命令

```bash
node src/0501-test.js
```

快捷键：`Ctrl + ``（反引号）` 打开终端 → 输入命令

### 2. Code Runner 插件（推荐）

1. 安装 **Code Runner** 插件
2. 快捷键：
   - `Ctrl + Alt + N` - 运行当前文件
   - `Ctrl + Alt + M` - 停止运行
3. 或者点击右上角 ▶ 按钮

### 3. VS Code 调试

1. 按 `F5` 选择 "Node.js"
2. 自动创建 `launch.json`
3. 设置断点后即可调试

### 4. 右键运行

安装 Code Runner 后，右键文件 → "Run Code"

## 代码编辑快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl + /` | 注释/取消注释 |
| `Alt + ↑/↓` | 移动代码行 |
| `Shift + Alt + ↓` | 复制代码行 |
| `Ctrl + D` | 选择相同单词 |
| `Ctrl + Shift + L` | 选择所有相同内容 |
| `Ctrl + .` | 快速修复 |
| `F2` | 重命名符号 |

## 文件操作

| 快捷键 | 功能 |
|--------|------|
| `Ctrl + N` | 新建文件 |
| `Ctrl + S` | 保存 |
| `Ctrl + P` | 快速打开文件 |
| `Ctrl + Shift + P` | 命令面板 |

## 光标移动

| 快捷键 | 功能 |
|--------|------|
| `Ctrl + ←/→` | 按单词移动 |
| `Home/End` | 行首/行尾 |
| `Ctrl + Home/End` | 文件头/文件尾 |

## 调试快捷键

| 快捷键 | 功能 |
|--------|------|
| `F5` | 开始调试 |
| `F9` | 切换断点 |
| `F10` | 单步跳过 |
| `F11` | 单步进入 |
| `Shift+F11` | 单步退出 |
| `Shift+F5` | 停止调试 |
